Manual to move from version v0.3.3 to v0.3.4

1- Execute in the root of the project:
* npm i grunt --save-dev
* npm i processpy --save-dev
* npm i grunt-contrib-processpy --save-dev

2- Add new file: Gruntfile.js

3- Add new item in "package.json" file.
"scripts": {
	"sql": "grunt process-sql"
},

4- Update "README.md" file.

5- Add new folder: "module/view/example".

6- Add new file: "module/controller/exampleController.php".

7- Add new file: "createDB.php".

Ok, updated project.
